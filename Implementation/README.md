# MicroMeet Web App

MicroMeet is a modern, demo-ready social meetup platform built with **Next.js (App Router)**, **React**, **TypeScript**, and **Tailwind CSS**.

This implementation uses **mock in-memory data** for users, meetups, participation, and notifications.

## Quick start

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Demo credentials

- `host1 / pass123`
- `user1 / pass123`
- `user2 / pass123`

## Main routes

- `/` - Login / landing
- `/dashboard` - Home dashboard
- `/profile` - Manage profile and preferences
- `/meetups` - Active / upcoming meetup feed
- `/meetups/[id]` - Meetup details
- `/meetups/create` - Create meetup
- `/meetups/edit` - Edit meetup
- `/participants` - Invite / approve participants
- `/responses` - Respond to join requests
- `/notifications` - Notifications center
